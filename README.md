# Flutter Posts App

A simple Flutter application that fetches and displays posts from an API.

## Setup

1. Add HTTP dependency:

```bash
flutter pub add http
```

2. Run the app:
```bash
flutter run
```

## API Reference
- Posts API: https://jsonplaceholder.typicode.com/posts
